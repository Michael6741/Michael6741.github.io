Oxy-Hydrogen Generator — Logo Design Process

A step-by-step build log from concept to export-ready assets.

Project Brief

Goal Create a modern, engineering-forward logo for an oxy-hydrogen (HHO) generator company, optimized for laser-etching, 3D printing, and small UI sizes.

Deliverables: SVG, PDF, PNG (1x/2x/4x), favicon + app icons
Constraints: 1-color friendly, high contrast, clean geometry, no tiny interior gaps
Concept keywords: hydrogen, spark/arc, molecule lattice, water → gas, efficiency, safety.

Research & Moodboard

Sketches → Direction

Three directions emerged: (A) Droplet + bolt, (B) Hex-cell with H-nodes, (C) Interlocking O-H ring. Direction B won for manufacturability and brand distinctiveness.

Vector Build

  • Built on a 12-pt grid with 60° increments; outer hex set by circumscribed circle diameter 180.
  • Node circles sized to survive laser-etch at 8px; minimum stroke 1.25px @ 24px icon.
  • All joins expanded + boolean-united for clean single path exports.
Path hygiene (click to expand)

Removed overlaps, rounded joins to 8px radius equivalents, normalized winding rules. Verified with outline mode and SVGOMG cleanup.

Iterations & Feedback

v0.2 — Node size too small

At favicon sizes nodes collapsed. Increased node diameter by 18% and thickened connectors by 0.25pt.

v0.6 — Droplet countershape

Introduced negative-space droplet in the center; improved category signaling without adding complexity.

v0.9 — Final geometry lock

Set final proportions; converted strokes to outlines; merged paths for single-color engraving.

Typography

  • Wordmark: Inter / Satoshi / SF Pro (SemiBold)
  • UI: System stack for performance
  • Spacing: Optical kerning; tracking −1% at display sizes

When paired with the mark, maintain a 1x cap-height gap between symbol and type.

Clear Space & Minimums

  • Clear space: 1× node diameter on all sides
  • Min size (screen): 20px height (mono), 28px (full color)

How to Recreate (Quick)

  1. Set a 12-pt isometric grid (60°) in your vector app.
  2. Draw outer hex; place six nodes on vertices.
  3. Connect nodes with rounded joins; create central negative droplet.
  4. Expand strokes, unite paths, remove overlaps.
  5. Export SVG (responsive), plus PNGs at 1×/2×/4×.

Apps used: Figma and Illustrator. Shortcuts: Ctrl/Cmd + ' (grid), Shift + drag (constrain angles).
